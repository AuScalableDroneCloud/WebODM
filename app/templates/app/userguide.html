User Guide - ASDC {{ version }}

Basics

To create a new project, press the "Add Project" button.
To create a map, press the "Upload Data" button. Then select "browse files" to select files, "browse folders" to select a folder or drag and drop images.
For further assistance, click the "User Guide" button on the menu to the left.

  • You need at least 5 images, but 16-32 is typically the minimum.
  • Images must overlap by 65% or more. Aim for 70-72%
  • For great 3D, images must overlap by 83%
  • GCP File is optional, but can increase georeferencing accuracy
